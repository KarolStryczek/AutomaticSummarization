class summarizers.LexRankSummarizer.LexRankSummarizer

Bases: summarizers.PageRankBasedSummarizer.PageRankBasedSummarizer

Implementation of LexRank algorithm.

It is based in summarization algorithm implemented in PageRankBasedSummarizer and implements only sentence similarity calculation function.

calculate_similarity(sentence_x: List[str], sentence_y: List[str]) → float

Sentence similarity implementation for LexRank algorithm. It is based on IDF modified cosine similarity of sentences.

Parameters:

  • sentence_x – Tokenized sentence

  • sentence_y – Tokenized sentence

Returns:

Similarity measure between sentences

get_idf(word: str) → float

Retrieves IDF value for word from dictionary calculated beforehand. If there is no IDF value for requested word, default value is returned. Default is calculated as max IDF value from dictionary.

Parameters:

word – Word for which IDF value should be returned

Returns:

IDF value for the word
