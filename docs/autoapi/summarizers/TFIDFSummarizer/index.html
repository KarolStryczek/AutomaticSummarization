Returns:

List of scores for sentences in the same order as input

get_idf(word: str) → float¶

Retrieve IDF value for word calculated beforehand. If word was not found in IDF dictionary, returns default IDF which is max IDF from IDF dictionary. Assumes that word that did not appear in source corpus 1 time is as common as the rarest word in corpus.

Parameters:

word – Word for which IDF value should be returned

Returns:

IDF value for word
