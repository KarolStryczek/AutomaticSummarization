get_csv_record() → Tuple[str, str, int, str, str, float, float, float, float, float, float, float, float, float]¶
Creates CSV record with columns:

  • filename

  • summary type

  • summary length to text length ratio

  • author

  • used summarizer

  • 9 columns of ROUGE-1, ROUGE-2, ROUGE-L - recall, precision and F-score

Returns:

Tuple with all necessary columns

class util.RougeEvaluation.RougeEvaluator¶

Class that enables ROUGE evaluation of text summarization algorithms on Polish Summaries Corpus. Contains necessary methods to load Polish Summaries Corpus, evaluate implemented algorithms and save results.

evaluate_all() → None¶

Calculates recall, precision and F-Score of ROUGE-1, ROUGE-2 and ROUGE-L for all reference summaries from Polish Summaries Corpus using all implemented summarizers. Calculated scores are saved after each text processed (each text have multiple reference summaries).

Returns:

None
